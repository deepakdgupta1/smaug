litellm is a Python SDK and proxy server (AI Gateway) that unifies access to 100+ LLM APIs including OpenAI, Anthropic, Azure, Bedrock, VertexAI, Cohere, and more. It provides a standardized OpenAI-compatible interface with built-in cost tracking, guardrails, load balancing, and comprehensive logging.

Essential for multi-model applications where you need to route API calls to different providers based on cost, latency, or availability, with unified monitoring and control.

## Key Features
- Call 100+ LLM APIs with unified interface
- Cost tracking and billing analysis
- Load balancing across multiple models
- Guardrails and safety controls
- Works as proxy server or Python SDK
- 46,200+ GitHub stars
