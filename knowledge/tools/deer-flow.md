## Description

A provider-agnostic guardrails system designed to be deterministic and useful for enterprise teams. It enables uniform enforcement of AI coding policies across multiple agents (Claude Code, Codex, etc.) while maintaining verifiable compliance at the PR level. Key for CTOs who need to enforce consistent safety policies across diverse AI coding tools.

## Key Features

- **Deterministic Guardrails**: Non-probabilistic policy enforcement
- **Provider Agnostic**: Works with Claude Code, Codex, and other coding agents
- **Policy Verification**: Check at PR level that guardrails were used and policies followed
- **Bypass Detection**: Ensures policies can't be circumvented
- **Team Enforcement**: Uniform guardrails for teams using multiple coding tools
