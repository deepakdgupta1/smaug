A web-based interface for the Hermes agent framework that brings conversation management, workspace file browsing, automated tasks, long-term memory, and multiple profiles directly into the browser.

## Key Features

- **Seamless Conversation Continuity**: Continue conversations from TUI and Telegram directly in the web interface
- **Workspace Profile Switching**: Instantly switch between working directories and profiles to avoid context pollution
- **Multi-Profile Management**: Organize multiple profiles as different "AI employees" with separate skills and memory
- **Pure UI Layer**: Lightweight frontend that connects to existing Hermes services without data migration
- **Remote Access Support**: SSH tunnel support for accessing Hermes services on remote servers
- **macOS Client Integration**: Works with dedicated Swift macOS client for enhanced experience (3MB footprint)
