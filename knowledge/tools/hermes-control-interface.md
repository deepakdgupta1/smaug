## Overview
A self-hosted web dashboard for managing Hermes Agent workflows and operations.

## Key Features
- **Browser-Based Terminal:** Direct terminal access in web UI
- **File Explorer:** Browse and manage agent files
- **Session Overview:** Monitor active sessions
- **Cron Management:** Schedule and manage recurring tasks
- **System Metrics:** Real-time monitoring of system resources
- **Agent Status Panel:** View status of multiple agents
- **Authentication:** Password-gated single-page access

## Use Cases
- Centralized management of multiple Hermes agents
- Long-running task monitoring
- Memory and workflow oversight
- Task scheduling and automation
