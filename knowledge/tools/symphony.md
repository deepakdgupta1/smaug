## Description

Symphony is an orchestration layer from OpenAI that polls project boards for changes and spawns agents for each lifecycle stage of a ticket. Instead of manually prompting agents to write code and create PRs, developers move tickets on a board and agents handle the rest.

## Key Features

- **Project Board Integration** - Automatically polls for ticket changes
- **Lifecycle Management** - Spawns specialized agents for each development stage
- **Board-Driven Workflow** - Move tickets instead of manually prompting agents
- **Automated Development** - Handles code writing, PR creation, and related tasks

## Concept

Replace manual agent prompting with visual workflow management. Developers interact with their existing project board interface, and agents react to state changes.
