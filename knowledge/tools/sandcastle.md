A TypeScript framework for orchestrating sandboxed coding agents. Built by Matt Pocock as part of his personal software factory. Provides a simple API for running coding agents in isolated environments with the core abstraction: `sandcastle.run()`.

## Key Features

- TypeScript-first design
- Sandboxed agent execution
- Simple, composable API (`sandcastle.run()`)
- Agent orchestration framework
- Production-ready implementation
