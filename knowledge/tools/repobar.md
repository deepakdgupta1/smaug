A macOS menubar application that brings GitHub repository status directly to your menu bar. Displays CI status, issue counts, pull request counts, and latest releases at a glance. Built in Swift for native performance and integration with macOS.

## Key Features
- Real-time GitHub repository status in menubar
- CI/CD status monitoring
- Issue and Pull Request tracking
- Latest release information
- Persistent SQLite caching to reduce API calls
- Rate limit visibility
- Archive fallback support
- Terminal integration
