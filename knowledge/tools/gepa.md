## Description
Auto-optimization tool for Claude system prompts (CLAUDE.md files) that uses execution traces and performance scores to improve model performance without model swaps. Tuning focused on instruction quality rather than model selection.

## Key Features
- Auto-optimizes prompt instructions with execution traces
- Scores and iterates on performance
- Targeted prompt tuning for CLAUDE.md files
- No model swap required
- Demonstrated 65% → 85% pass rate improvement on Haiku 4.5
- Applicable to any Claude Code builder workflow
