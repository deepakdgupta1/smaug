Persistent memory for Claude Code (CPR: Compress, Preserve & Resume). A set of skills to save key learnings, capture full sessions, and restore context, significantly reducing token costs for session restarts.

## Key Features

- /preserve: Save key learnings to CLAUDE.md
- /compress: Capture full session to a searchable log
- /resume: Restore context in new sessions
