Browser Harness is an open-source, lightweight (~600 lines) tool from the Browser Use team. It connects LLMs directly to a real Chrome browser via raw Chrome DevTools Protocol (CDP) with zero heavy frameworks or abstractions. The agent gets full freedom and can self-heal by writing missing helper functions at runtime.

Perfect for reliable web tasks in agents like Hermes—avoids bot blocks and improves speed vs traditional automation.

## Key Features

- Lightweight implementation (~600 lines)
- Direct Chrome DevTools Protocol (CDP) integration
- Zero heavy frameworks or abstractions
- Self-healing capabilities (agents can write helper functions at runtime)
- Optimized for LLM-driven automation
- Faster and more reliable than traditional browser automation
- Bot-detection evasion
