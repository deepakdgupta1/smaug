LLMLingua is a prompt compression framework that reduces prompt size by up to 95%, achieving up to 20x compression with near-zero quality loss. Published at EMNLP 2023 and ACL 2024, it enables faster LLM inference and improves models' perception of key information by compressing both prompts and KV-Cache.

The tool is particularly valuable for reducing token consumption in LLM-based systems, making it cost-effective for production deployments where prompt size directly impacts inference costs and latency.

## Key Features
- Up to 95% prompt compression
- 20x compression ratio with minimal performance loss
- Compresses prompts before API calls
- Published research with comprehensive evaluation
- 6,100+ GitHub stars
