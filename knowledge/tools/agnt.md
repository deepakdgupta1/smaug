## Description

AGNT is a self-hosted desktop environment for building and running AI agents. It provides a local-first platform where developers can create, test, and deploy agents without relying on cloud services.

## Key Features

- Self-hosted desktop environment
- Agent building and execution capabilities
- Local-first architecture for privacy and control
- JavaScript/Node.js based
- Active development community (262+ stars)
