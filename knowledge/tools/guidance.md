Guidance is a guidance language for controlling large language models. It provides a framework for constraining and directing LLM outputs to follow specific patterns, formats, and logical flows, enabling more reliable and structured outputs from language models.

Essential for applications requiring deterministic or structured outputs from LLMs without relying on unreliable parsing.

## Key Features
- Control and constrain LLM outputs
- Structured output guidance
- Language-based control primitives
