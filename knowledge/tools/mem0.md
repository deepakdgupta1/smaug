mem0 is a universal memory layer for AI agents that replaces full conversation history with intelligent memory retrieval. It achieves 50x reduction in token consumption (10,000 token history → 200 token memory per agent) by storing only what matters and retrieving only what's needed in context.

This tool is essential for building stateful AI agents that can maintain context across sessions without exploding token usage, making it practical for production agent systems.

## Key Features
- Replaces full conversation history in context
- 50x reduction in memory tokens (10,000 → 200 tokens)
- Works as a universal layer across agent types
- 55,000+ GitHub stars
- Supports multiple memory backends
