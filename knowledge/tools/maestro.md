## Overview
Conductor for cross-agent coding workflows. Maestro coordinates multiple AI agents (Claude Code, Codex, Hermes) with structured memory, handoffs, and evidence-based trust mechanisms.

## Key Features
- **Cross-Agent Orchestration:** Coordinate multiple AI agents in workflows
- **Structured Memory:** Persistent state across agent handoffs
- **Plan-Approve-Execute:** Explicit workflow control pattern
- **Trust Substrate:** Evidence-based agent evaluation and trust scoring
- **External CLI Support:** Integrates with Codex, Claude Code, and Hermes

## Architecture
- Agent coordination layer
- Memory management system
- Trust and verification framework
- Workflow planning and execution
- MCP plugin support

## Use Cases
- Multi-agent code generation
- Complex project development
- Agent skill combination
- Workflow orchestration
- Agent evaluation and monitoring
