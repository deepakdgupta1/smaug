gstack - An enhanced engineering review and security analysis tool for Claude Code

An MIT-licensed tool that integrates with Claude Code to provide comprehensive code review capabilities, including subtle security vulnerability detection such as cross-site scripting attacks.

## Key Features

- Advanced code review and analysis
- Cross-site scripting (XSS) detection
- Integration with Claude Code
- Open source (MIT license)
- Simple one-paste installation
