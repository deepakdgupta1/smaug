## Description
Open-source desktop app that replicates Bloomberg Terminal functionality at a fraction of the cost. Runs on Windows, macOS, and Linux with CFA-level analytics, investor AI agents, and extensive data connectors.

## Key Features
- Full CFA Level 1, 2, 3 analytics implemented in Python
- 20+ investor AI agents (Buffett, Dalio, Graham, Soros, Lynch, Klarman personas)
- Hedge fund presets (Bridgewater All-Weather, Citadel multi-strat, Renaissance stat models)
- 100+ data connectors including Polygon, Kraken, DBnomics (100M+ series), World Bank
- Cross-platform: Windows, macOS, Linux
- Alternative to $24,000/year Bloomberg Terminal
