## Overview
Hermes Atlas is a community map of every tool, skill, and integration available for Hermes Agent. Live at hermesatlas.com.

## Key Features
- **Comprehensive Map:** Catalog of 100+ tools and integrations
- **Visual Exploration:** Interactive browsing of the Hermes ecosystem
- **RAG Search:** AI-powered search across tools and skills
- **Community Contributions:** Growing collection of community resources

Provides a single source of truth for discovering and understanding all tools, skills, and integrations available in the Hermes Agent ecosystem.
