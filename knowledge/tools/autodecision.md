## Description
Claude Code/Cowork plugin that applies Karpathy's LLM Council pattern with autoresearch to business and strategy decisions. Moves beyond simple ChatGPT iteration by creating a council of perspectives to stress-test decisions.

## Key Features
- Pulls real data for informed decisions
- Runs council of POVs: optimist, pessimist, market, competition, regulator
- Models second-order effects
- Stress-tests worst cases and black swan scenarios
- Identifies stable vs weak effects
- Iterates until decisions hold up under scrutiny
- Built as Claude Code plugin for seamless integration
