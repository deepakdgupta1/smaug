## Overview
Hermes is a self-evolving AI agent framework that learns from experience and automatically creates its own skills. The agent maintains memory across sessions and builds reusable skills from workflows.

## Key Features
- **Self-Learning:** Agent watches workflows and automatically generates new skills
- **Memory Persistence:** Sessions remember context and experience across restarts
- **Skill Factory:** Automatic skill creation from repeated tasks
- **Multi-Agent Support:** Control interface for managing multiple agents
- **Community Ecosystem:** 100+ integrated tools and skills

## Core Components
- Hermes Agent (main engine)
- Control Interface (dashboard for agent management)
- Skill Factory (automatic skill generation)
- Hermes-Wiki (implementation documentation)

## Ecosystem Tools
- **Atlas/Hermes Ecosystem:** Community map of 100+ tools and integrations
- **Maestro:** Cross-agent orchestration and workflow coordination
- **Hermes HUD:** TUI consciousness monitor
- **Awesome Hermes Agent:** Curated list of skills and resources
