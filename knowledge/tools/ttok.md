ttok is a utility for counting and truncating text based on tokens. Provides a simple interface for token-aware text processing, enabling applications to work with token-constrained limits without losing important content.

A practical tool for managing token budgets when working with length-limited LLM APIs.

## Key Features
- Count tokens in text
- Truncate text to token limits
- Token-aware text processing
