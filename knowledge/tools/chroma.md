Chroma is search infrastructure for AI, providing a modern vector database built in Rust. It enables AI agents and applications to store and retrieve semantic information efficiently, supporting similarity search and retrieval-augmented generation workflows.

A foundational tool for any AI system that needs to manage embeddings and perform semantic search across knowledge bases.

## Key Features
- Vector database built in Rust
- Semantic search capabilities
- Support for AI agents and applications
- Efficient embeddings storage
- 27,800+ GitHub stars
