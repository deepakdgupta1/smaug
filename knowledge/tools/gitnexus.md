## Description

GitNexus is an open-source knowledge graph engine for codebases. It's not a documentation tool - it's a full code intelligence system that builds a structured graph of your codebase relationships, dependencies, and interactions.

## Key Features

- Knowledge graph construction for codebases
- Code intelligence and relationship mapping
- Enables AI agents to understand codebase structure deeply
- Supports code search and navigation
- Foundation for building intelligent code understanding tools
