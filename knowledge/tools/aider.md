aider is AI pair programming in your terminal. It enables developers to write and modify code collaboratively with AI models (Claude 3, GPT-4, Gemini, and others) directly from the command line, supporting multiple models with seamless switching.

A practical tool for integrating AI-assisted coding into existing development workflows without leaving the terminal.

## Key Features
- AI pair programming in terminal
- Multiple model support (Claude, GPT-4, Gemini, Llama)
- Code generation and modification
- CLI-based workflow
