## Overview
A meta-skill plugin for Nous Research's Hermes AI agent that automatically watches workflows and converts them into reusable skills.

## Problem Statement
When working with Hermes, completed workflows disappear at the end of each session. Users have to re-explain the same solution every time they encounter the same problem. This wastes time and loses institutional knowledge.

## Solution
Skill Factory automatically:
- Watches user workflows during sessions
- Captures problem-solution patterns
- Converts workflows into reusable skills
- Persists skills across sessions

## Common Workflow Examples
- Setting up new projects
- Debugging code
- Creating pull requests
- Configuration management
- Testing procedures

## Benefits
- **Efficiency:** Reuse solutions without re-explanation
- **Knowledge Capture:** Preserve workflows as skills
- **Learning:** Agents improve over time
- **Scaling:** Skills can be shared across agent instances
