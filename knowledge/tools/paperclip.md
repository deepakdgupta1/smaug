## Description

Paperclip is an open-source orchestration layer for building zero-human companies. It provides everything needed to run autonomous businesses with a focus on design quality and user experience comparable to products like Linear.

## Key Features

- **Org Charts** - Define organizational structure for autonomous systems
- **Goal Alignment** - Align agent actions with business objectives
- **Task Ownership** - Assign and track autonomous task execution
- **Budget Management** - Control costs and resource allocation across agents
- **Agent Templates** - Pre-built templates for common automation patterns
- **No Vendor Lock-in** - Not tied to any specific agent provider
