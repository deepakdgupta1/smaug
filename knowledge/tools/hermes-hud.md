## Overview
A terminal user interface (TUI) consciousness monitor for Hermes Agent. Provides real-time visibility into agent state, memory, and operations from the command line.

## Key Features
- **Real-Time Monitoring:** Live agent state updates
- **TUI Interface:** Terminal-based dashboard
- **Status Visualization:** Agent awareness and metrics
- **System Integration:** Works with existing Hermes deployments
- **Lightweight:** Pure Python implementation

## Technology Stack
- **Framework:** Textual (Python TUI framework)
- **Inspiration:** neofetch-style system monitoring
- **Terminal:** Full terminal environment for monitoring

## Use Cases
- Development and debugging
- Agent health monitoring
- Workflow visibility
- Remote agent oversight
- System performance tracking
