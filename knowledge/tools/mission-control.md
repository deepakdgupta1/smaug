Dashboard for AI agent orchestration with real-time monitoring, cost tracking, and multi-gateway support. Open-source project featuring 26 panels, WebSocket + SSE updates, SQLite backend (no external services), role-based access, quality gates, and Kanban board.

## Key Features

- 26 monitoring panels
- Real-time WebSocket and SSE updates
- SQLite database (zero external service dependencies)
- Kanban board for task tracking
- Cost tracking and analytics
- Role-based access control
- Quality gates for agent outputs
- Multi-gateway support
- Single command startup (`pnpm start`)
