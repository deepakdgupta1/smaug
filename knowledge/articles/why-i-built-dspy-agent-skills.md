An explanation of why a developer created a specialized agent skills package for DSPy, bridging the gap between DSPy's research foundations and practical agent implementation.

## Key Takeaways

- **The Knowledge Gap**: Despite excellent documentation, Claude Code and Codex CLI struggle with idiomatic DSPy code and frequently use deprecated imports or runtime-crashing patterns
- **Solution: Curated Skills**: dspy-agent-skills packages five Agent Skills that give coding agents working knowledge of idiomatic DSPy, similar to onboarding a new teammate
- **Three Core Insights**: DSPy is about programming, not prompting; GEPA enables optimization without RL; RLM handles inputs larger than context windows
- **Ecosystem Alignment**: Built to complement the growing Agent Skills specification, compatible across Claude Code and Codex CLI
- **Real-World Testing**: Validated against current Claude Code and OpenAI Codex CLI implementations
