## Summary

The problem: AI agents struggle to find and organize files within knowledge bases. The solution: add YAML metadata (breadcrumbs) to all files and teach agents to follow those breadcrumbs while searching data.

This pattern emerged from the successful approach of having AI agents and notes live in the same folder (as shown in Obsidian second brain implementations). By adding structured YAML frontmatter with navigation metadata, agents can efficiently locate and work with the right files.

## Key Takeaways

- AI agents can't inherently navigate your file structure efficiently
- YAML frontmatter acts as "breadcrumbs" for agent navigation
- Structured metadata in files enables agents to follow logical paths
- This pattern integrates well with Obsidian vaults and note-taking systems
- Enables faster, more intelligent file discovery by AI agents
- Simple solution: metadata + agent instructions = better file interaction
