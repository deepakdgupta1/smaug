Deep dive into building production-ready agentic software. Ashpreet argues that building the agent is the easy part; the hard part is the runtime (the harness) that handles durability, state, and scalability.

## Key Takeaways

- Durability: Agents reasoning across multiple steps need a distributed systems approach.
- State management: Maintaining context across sessions is critical.
- Serving: Horizontally scalable agent APIs are the next frontier.
