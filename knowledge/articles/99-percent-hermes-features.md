Most Hermes users exploit only 8% of the platform's capabilities. The article reveals 15 features ranked by impact that 99% of users haven't touched: personality switching, persistent memory files (MEMORY.md, USER.md), cross-session analytics, file rollbacks, voice mode, 17-platform reach, custom slash commands, and more.

Hermes is a provider-agnostic AI agent with 100+ pre-built skills that runs on 17 messengers from one process and can swap models mid-session without restarting. Most users treat it like ChatGPT instead of leveraging its full architecture.

The setup features users skip include SOUL.md (persistent personality definition), MEMORY.md and USER.md (cross-session context), /personality command for persona switching, /insights for token analytics, and file versioning/rollback capabilities.

## Key Takeaways
- Hermes has 92% of features most users never discover
- Persistent memory files (SOUL.md, MEMORY.md, USER.md) are foundational
